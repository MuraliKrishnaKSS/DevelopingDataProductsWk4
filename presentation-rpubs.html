About the project

  • The project is part of week#4's assignment of Developing Data Products course to demonstrate skills in Shiny and Rstudio Presenter.
  • Shiny is used to prepare an application and Rstudio Presenter is used to prepare html5 deck of slides as part of the project.
  • Shiny application demonstrates ui and server interactions. It uses fluidpage with sidebarlayout along with sidebarpanel and mainpanel, radioButtons and selectInput control widgets, textOutput and plotlyOutput functions, renderText and renderPlotly render functions.
  • A data from a “csv” file is used to populate plots (charts) in the application. Plots are created using Plotly package.
  • User can select different values using the input control widgets and the System presents appropriate data in the form of a scatter plot and pie chart.
  • Slide deck explains the project and the application

Deforestation and 30,979 threatened life forms


World-wide threatened species are grouped under birds, fish, mammals and plants

  • 30,979 species can soon become extinct if due care is not taken. Population pressures, deforestation and climate change are major causes.
  • Content of this application is drawn from data table called World Development Indicators: Deforestation and biodiversity (“http://wdi.worldbank.org/table/3.4”)
  • Dataset was reshaped, country metadata was added and a subset suitable for the current application was selected.

Top 30 account for 54.55%

  • Top Ten Countries account for 9677 of threatened species (31.23%) ::
  • These ten countries are depicted on the map.
  • The next 20 countries account for 7223 more species.
  • Together the top 30 countries account for 30979 threatened species (54.55%).

Application user interface

Data from 214 countries can be viewed. They are grouped to reduce clutter. Forest growth rate (-ve) is deforestation (+ve) is afforestation. (application )

  1. Select grouping criteria : Income or Region.
  2. Select sub-group. Options change by grouping criteria.
  3. Select any specific life form or “All”.
  4. View aggregate data of sub-group as a pie-chart
  5. Title changes to suit plotted data.
  6. Filtered data is plotted. View country-wise deforestation (-ve) / afforestation (+ve) rates (X axis) and number of threatened species (Y axis). Hover mouse over markers for details.
